# python-nmap-port-scanner

A simple cybersecurity tool written in Python that automates port scanning using Nmap.

## Features
- Takes a target IP or domain as input
- Runs an Nmap service detection scan (-sV)
- Displays results in the terminal
- Saves output to scan_report.txt

## Requirements
- Python 3
- Nmap installed

## Usage
python3 scanner.py

Enter a target like:
scanme.nmap.org
